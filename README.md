# CODTECH Java Internship Projects

## Overview
This repository contains all four Java internship tasks completed as part of the CODTECH Internship Program.

The projects demonstrate core Java concepts including:
- File Handling
- REST API Integration
- Socket Programming
- Multithreading
- AI-Based Recommendation Logic

Each task is implemented using beginner-friendly Java code with proper comments and explanations.

# Tasks Included

## Task 1 - File Handling Utility

### Description
A Java program that performs basic file operations such as:
- Writing data into a file
- Reading data from a file
- Modifying existing file content

### Technologies Used
- Java File Handling
- FileReader
- FileWriter
- BufferedReader

### File
```text
FileHandlingUtility.java
```

## Task 2 - REST API Client

### Description
A Java application that connects to a public REST API, fetches JSON data, and displays structured information.

### API Used
```text
https://api.agify.io
```

### Technologies Used
- Java Networking
- HttpURLConnection
- JSON Parsing

### File
```text
RestApiClient.java
```

## Task 3 - Multithreaded Chat Application

### Description
A client-server chat application using Java Sockets and Multithreading.

Features:
- Multiple client handling
- Real-time communication
- Separate threads for each client

### Technologies Used
- Socket Programming
- Multithreading
- DataInputStream
- DataOutputStream

### Files
```text
ChatServer.java
ChatClient.java
```
