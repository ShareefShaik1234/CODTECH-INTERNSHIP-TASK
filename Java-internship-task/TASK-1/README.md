# File Handling Utility in Java

## Overview
This project is a Java-based File Handling Utility developed as part of the CODTECH Internship Task 1.

The application allows users to:
- Write data into a text file
- Read data from a text file
- Modify existing file content

The program is menu-driven and beginner-friendly.

---

## Features
- Write data to file
- Read file content
- Modify file content
- Exception handling
- User-friendly console interface

---

## Technologies Used
- Java
- File Handling
- Exception Handling
- Scanner Class
