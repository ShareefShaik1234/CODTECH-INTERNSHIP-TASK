# AI-Based Recommendation System in Java

## Overview
This project is an AI-Based Recommendation System developed for CODTECH Internship Task 4.

The system recommends products based on user-selected categories.

Categories Included:
- Electronics
- Books
- Fashion
- Food

---

## Features
- Product recommendation engine
- User preference handling
- Category-wise suggestions
- Sample dataset included
- Beginner-friendly implementation

---

## Technologies Used
- Java
- Collections Framework
- HashMap
- ArrayList

---

```bash
RecommendationSystem.java
```
