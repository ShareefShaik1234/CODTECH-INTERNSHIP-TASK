# Multithreaded Chat Application in Java

## Overview
This project is a Java-based Multithreaded Chat Application developed for CODTECH Internship Task 3.

The application uses:
- Java Socket Programming
- Multithreading
- Client-Server Architecture

Multiple clients can connect to the server and chat in real time.

---

## Features
- Real-time messaging
- Multiple client handling
- Separate thread for each client
- Client-server communication
- Socket programming

---

## Technologies Used
- Java
- Socket Programming
- Multithreading
- DataInputStream
- DataOutputStream

---

## Files Included

```text
ChatServer.java
ChatClient.java
```

---

## How to Compile

```bash
javac ChatServer.java
javac ChatClient.java
```

---

## How to Run Server

```bash
java ChatServer
```

---

## How to Run Client

Open another terminal and run:

```bash
java ChatClient
```

You can open multiple clients for group chat.
